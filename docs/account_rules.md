# Правила форматирования счетов и реквизитов (Черногория)

## Номера счетов

### Структура: BBB-NNNNNNNNNNNNN-CC (3-13-2 = 18 цифр)

- **BBB** (3 цифры) — код банка (всегда первые 3 цифры)
- **NNNNNNNNNNNNN** (13 цифр) — номер счёта (дополняется нулями слева если короче)
- **CC** (2 цифры) — контрольное число

### Типы счетов (по первой цифре после кода банка)

- **5xx** — клиентские счета (банковские)
- **8xx** — государственные счета

### Форматы

| Формат | Пример | Описание |
|--------|--------|----------|
| Короткий с дефисами | `535-22023-67` | BBB-NNN...N-CC, средняя часть может быть короткой |
| Полный с дефисами | `570-1110011238-10` | BBB-NNNNNNNNNNNNN-CC |
| 18 цифр без разделителей | `520000000004307069` | Целевой формат |
| IBAN | `ME25520000000004307069` | ME25 + 18 цифр |

### Правила нормализации

1. Если есть префикс `ME25` — убираем его, остаются 18 цифр
2. Если формат с дефисами `BBB-N...N-CC` — убираем дефисы, среднюю часть дополняем нулями слева до 13 цифр
3. Если между частями есть пробелы или другие разделители — убираем, дополняем нулями
4. **Итоговый формат всегда: 18 цифр** (или `ME25` + 18 цифр для IBAN)

### Примеры нормализации

```
535-22023-67         → 535000000002202367
570-1110011238-10    → 570011100112381­0  → 570111001123810  (нет, правильно: 570-1110011238-10 → 5701110011238010)
ME25 520 0000000043 07069 → 520000000004307069
ME25520000000004307069    → 520000000004307069  (IBAN → 18 цифр)
```

## PIB (Порески идентификациони број)

- **Всегда 8 цифр**
- Если в документе меньше 8 цифр — дополняем нулями слева
- Пример: `3339645` → `03339645`

## Применение в экспорте 1С

- `РасчСчет` — 18 цифр без дефисов
- `ПлательщикСчет` / `ПолучательСчет` — 18 цифр без дефисов
- `ПлательщикИНН` / `ПолучательИНН` — PIB 8 цифр с ведущими нулями
